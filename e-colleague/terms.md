# e-colleague 利用規約

## 第1条 定義

本規約において、以下の用語は次の意味を有します。

| 用語 | 定義 |
|------|------|
| **本サービス** | 開発者Shimataiyakiが提供するLINE Bot「e-colleague」および関連する管理者ポータル、スプレッドシート連携機能の総称 |
| **OPS-0** | 本サービスの背後にある設計フレームワーク。チームの「気がつく人」に負荷が集中する構造を解決するための指針群 |
| **ユーザー** | 本サービスが導入されたLINEグループに所属し、本サービスを利用する個人 |
| **グループ管理者** | プロダクトキーを保有し、本サービスをLINEグループに導入した者 |
| **プロダクトキー** | 本サービスをLINEグループで利用可能にするために必要な認証コード |
| **コンテンツ** | ユーザーが本サービスを通じて送信するメッセージ、タスク情報その他のデータ |
| **本規約** | このe-colleague利用規約 |

## 第2条 本規約への同意

1. グループ管理者は、LINEグループにおいてプロダクトキーを送信し認証を受けた時点で、本規約に同意したものとみなされます。
2. ユーザーは、本サービスが導入されたLINEグループにおいて本サービスを利用した時点で、本規約に同意したものとみなされます。
3. 本規約に同意できない場合は、本サービスを利用することはできません。

## 第3条 サービスの概要

1. 本サービスは、LINEグループ上の会話からタスクを抽出し、Googleスプレッドシートに自動記録するタスク管理アシスタントです。
2. 本サービスは「e-colleague」の名称で提供され、LINE Messaging API、Google Apps Script、Gemini APIを用いて動作します。
3. 本サービスはOPS-0の設計思想に基づき、「情報はチームの共有財産である」という指針のもと、チーム内の認識共有を整理するために設計されています。
4. 本サービスは現在ベータ版として提供されており、予告なく仕様変更される場合があります。

## 第4条 利用条件

1. 本サービスは、プロダクトキーによる認証を受けたLINEグループ内でのみ利用可能です。
2. 個人トーク（1対1のトーク）での利用はできません。
3. 本サービスの利用には、LINEアカウントが必要です。
4. 13歳未満の方が利用する場合は、保護者の同意が必要です。

## 第5条 禁止事項

ユーザーは、本サービスの利用にあたり、以下の行為を行ってはなりません。

1. 法令または公序良俗に違反する行為
2. 他者の権利（知的財産権、プライバシー権、名誉権等）を侵害する行為
3. Botに対して過剰な負荷をかける行為（短時間に大量のメッセージを送信する等）
4. プロダクトキーを不正に共有・再配布する行為
5. スパムまたは荒らし行為
6. 本サービスの運営を妨害する行為
7. その他、開発者が不適切と判断する行為

## 第6条 免責事項

1. 本サービスは現状有姿（as-is）で提供され、特定の目的への適合性、有用性、正確性、完全性について一切の保証を行いません。
2. 本サービスの利用により生じたいかなる損害（データの消失、業務の中断、間接損害、特別損害を含む）について、開発者は一切の責任を負いません。
3. タスク抽出・判定には誤認識や誤判定が含まれる可能性があり、これにより生じた損害について開発者は責任を負いません。
4. 本サービスが利用する外部サービス（LINE Messaging API、Gemini API、Googleスプレッドシート等）の障害・停止により本サービスが利用できなくなった場合も、開発者は一切の責任を負いません。
5. ユーザーが本サービスを通じて送信したコンテンツについては、ユーザー自身の責任において管理されるものとします。

## 第7条 サービスの変更・中断・終了

1. 開発者は、事前の通知なく本サービスの内容を変更、中断、または終了することができます。
2. 本サービスの変更、中断、終了によりユーザーに生じた損害について、開発者は一切の責任を負いません。

## 第8条 準拠法および管轄裁判所

1. 本規約の解釈および適用は、日本法に準拠します。
2. 本サービスに関連して生じた紛争については、開発者の居住地を管轄する裁判所を第一審の専属的合意管轄裁判所とします。
