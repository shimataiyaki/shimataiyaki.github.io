自己紹介

Shimataiyaki として活動する高校生。デジタルとアナログの接点、あるいは日常にちょっとした遊び心を加えるプロダクトを制作しています。

主な活動領域はウェブフロントエンド、インタラクションデザイン、そしてオフラインでの経験をソフトウェアに落とし込むことです。

GitHub @shimataiyaki

プロダクト・作品

各プロダクトの詳細なコードやドキュメントは、GitHubリポジトリをご参照ください。

デジタル絵馬 digital-emaプロダクト

入力した願い事がモニターに絵馬として表示される、文化祭向け参加型インスタレーション。最大6件まで同時掲示され、定期的に自動更新されます。

GitHub: digital-ema

  • 利用技術：HTML/CSS, JavaScript
  • 特徴：GAS利用、表示ローテーション

二進数おみくじ binary-omikujiプロダクト

画面に二進数のみを表示し、対応する紙のおみくじカードと引き換えることで結果が分かる、アナログ連動型の運試しジェネレーター。

GitHub: binary-omikuji

  • 利用技術：HTML/CSS, JavaScript (乱数生成)
  • 特徴：SNS連携 (#二進数おみくじ)、リアルイベント前提設計

Hama-Portal 浜松市紹介サイト作品

運用終了

中学3年次の「総合的な学習の時間」の一貫として制作した、地元・浜松市の魅力を発信するポータルサイト。観光スポットやグルメ情報をまとめ、実際に一般公開していました。

  • 運用期間：2023年9月 〜 2025年1月
  • 更新頻度：初期は月1回程度、以降は高校受験に専念するため更新を停止
  • 位置づけ：学校の探究学習の一環として企画・開発・公開までを完結

※ 現在はサービスを終了しており、公開アーカイブはありません。
