ポートフォリオ

最終更新：2026年4月18日
デジタルとアナログの接点に、遊び心を仕込む。

Shimataiyaki として活動する高校生。デジタルとアナログの境界、日常に潜むちょっとした違和感や遊びをプロダクトに落とし込んでいます。オフラインの体験をデジタルで拡張すること、あるいはデジタルをきっかけにオフラインの交流が生まれることに興味があります。

GitHub @shimataiyaki

タイムライン

2023

地元・浜松市の魅力を伝えたいという思いから、初めてウェブサイトを公開。デジタルを通じてオフラインの価値を拡張することに興味を持つ。

2025

国際交流の現場で、言語の壁を超える交流の必要性を実感。コンテストにも精力的に応募し、実績を築く。

2026

これまでの実績を経て、個人開発を本格化。「体験をデザインする」ことを軸に、複数のプロジェクトを展開中。

制作物

デジタル絵馬 digital-ema

入力した願い事がモニターに絵馬として表示される、文化祭向け参加型インスタレーション。最大6件まで同時掲示され、定期的に自動更新。

GitHub
二進数おみくじ binary-omikuji

画面に二進数のみを表示し、対応する紙のおみくじカードと引き換えることで結果が分かる、アナログ連動型の運試しジェネレーター。

GitHub
Hama-Portal 浜松市紹介サイト 運用終了

中学3年次の「総合的な学習の時間」で制作した、地元・浜松市の魅力を発信するポータルサイト。観光スポットやグルメ情報をまとめ、一般公開していた。
