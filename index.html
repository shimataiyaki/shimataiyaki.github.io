About

自己紹介

最終更新：2026年5月11日

Shimataiyaki として活動する高校生。日々の様々な活動からインスピレーションを得て、文章やプロダクトに落とし込んでいます。

GitHub @shimataiyaki

Story

タイムライン

2023

地元・浜松市の魅力を伝えたいという思いから、初めてウェブサイトを公開。デジタルを通じてオフラインの価値を拡張することに興味を持つ。

2024

第25回全国スカウトフォーラムに静岡県連盟代表として参加。ベンチャースカウトの行動の指針提言に議長団の一員として携わる。

2025

国際交流の現場で、言語の壁を超える交流の必要性を実感。コンテストにも精力的に応募し、実績を築く。

2026

これまでの実績を経て、個人開発を本格化。「体験をデザインする」ことを軸に、複数のプロジェクトを展開中。

Works

制作物

2023

Hama-Portal

運用終了

中学3年次に制作した、地元・浜松市の魅力を発信するポータルサイト。「自分の好きな街を、自分で紹介したい」という思いから生まれた、初めてのWebサイトです。

※ 現在は公開アーカイブはありません。

E-LOTs

二進数おみくじ

二進数だけを画面に表示し、紙のおみくじカードと引き換えるアナログ連動型ジェネレーター。デジタルの無機質さと、紙を手渡す温かみのギャップを楽しんでほしい。

GitHub

E-MAs

デジタル絵馬

願い事をモニターに絵馬として表示する、文化祭向け参加型インスタレーション。「デジタル神社」で参拝者が思いを共有できる場をつくりたくて生まれました。

GitHub

Product 04

Mendako BBS

シンプルな掲示板アプリ。余計な機能を削ぎ落とし、「書く・読む・返す」だけに集中できる設計。個人サーバーで動作する軽量さが特長です。

GitHub

Product 05

e-colleague

LINEグループに話しかけるだけでAIがタスクを抽出・管理するBot。口頭の「やっといて」を自動でタスクに変え、チームの「言った言わない」をなくします。
