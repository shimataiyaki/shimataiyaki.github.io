全プロダクト共通規約

以下の条項は、Shimataiyaki が提供するすべてのプロダクトに共通して適用されます。

「本プロダクト」とは、Shimataiyaki が制作したデジタル絵馬、二進数おみくじ、および関連するソフトウェア群を指します。

  • 本プロダクトのソースコードは、特に明記がない限り MITライセンス に基づき公開されています。
  • 詳細は各プロダクトの GitHub リポジトリをご確認ください。
    • 🔗 デジタル絵馬: github.com/shimataiyaki/digital-ema

表示について

  • ご入力いただいた「ニックネーム」と「願い事」は、接続されたモニターに絵馬として表示されます。

二進数おみくじ ご利用にあたって

以下の規約は「二進数おみくじ」固有の取り決めです。上記の共通規約とあわせてお読みください。

運営者

Shimataiyaki

おみくじのしくみ

  • 画面上では「二進数（例：0110）」のみが表示されます。
  • 表示された数字を展示ブースの部員に見せることで、対応する紙のおみくじカードを受け取ることができます。
  • 画面上に運勢（大吉など）は表示されません。紙カードと引き換えることで初めて結果が判明します。

ご利用にあたってのお願い

  • 本プロダクトは無料でご利用いただけます。
  • 連続してお引きいただくことは可能ですが、多くの方に楽しんでいただくため、譲り合ってご利用ください。
  • 表示された二進数のスクリーンショットをSNS等に投稿する際は、ハッシュタグ #二進数おみくじ をつけていただけると幸いです。

お問い合わせ

  • ご意見・ご感想は、GitHub の Issue よりお寄せください。
    https://github.com/shimataiyaki/binary-omikuji/issues
